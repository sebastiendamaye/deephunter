Scripts
#######

Description
***********

DeepHunter is shipped with several scripts that help automate tasks such as backups, running campaigns, optimizing the database, and upgrading the software.

These scripts are designed to be run from the command line and can be scheduled using cron jobs.

Location
********

Scripts are located in the  ``./qm/scripts/`` folder.
